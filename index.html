abstract

SEVEN is an energy-aware routing experiment. Instead of sending every prompt to the same large model, it classifies the prompt first and chooses a local-class, direct-data, or cloud-class route.

system

The full CLI can connect to Lemonade Server for local inference. This browser version keeps the interface portfolio-safe by simulating the local path with a small hosted model and the cloud path with a stronger hosted model.

routing

Short factual prompts start local. Current-data prompts are marked for direct data lookup. Long, specialized, or complex prompts escalate to cloud-class reasoning.

energy

The watt-hour values are estimates, not hardware measurements. SEVEN compares the selected route against a cloud-only baseline so the cost of routing decisions stays visible.
